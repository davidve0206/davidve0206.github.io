Evolution of NBA Rebounding

EDA project using SQL and Python (Pandas, Plotly, Dash)

Project to practice EDA using SQL and Python by exploring how rebounding, particularly from players in the "Guard" positions, has evolver in the NBA over time. This is done mostly by visualizing the data through Plotly, and the data can be easily visualized through a simple dashboard, created using Dash.
