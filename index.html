Evolution of NBA Rebounding

EDA project using SQL and Python (Pandas, Pyplot)

Project to practice EDA using SQL and Python by exploring how rebounding, particularly from players in the "Guard" positions, has evolver in the NBA over time. This is done mostly by visualizing the data through Plotly, and will later be accompanied by a dashboard using Dash.
